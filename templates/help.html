How it works

  1. Create or select a subject

    Subjects are folders under summry_web/data/subjects/. Use the sidebar tabs or New subject.

  2. Upload a PDF

    The app extracts text, builds a structured summary, flashcards, and a 10-question practice test.

  3. Open the study pack

    After processing, the page refreshes and the pack appears under that subject. Click it to read the full HTML study page.

What each study pack contains

  • Summary — TL;DR, key concepts, definitions, testable points, common confusions
  • Scenarios — “When to use X vs Y” practice
  • Flashcards — click to flip, tagged Core / Detail
  • Practice test — multiple choice and true/false with answer key

Run the app

cd summry_web
pip install -r requirements.txt
python app.py

Then open http://127.0.0.1:5000/

Run tests

cd summry_web
python -m pytest

See README.md for full documentation.

Folder layout (self-contained)

summry_web/
  app.py              # web server
  pipeline/           # extract, summarize, generate
  templates/          # hub, help, new subject pages
  static/assets/      # study pack CSS & JS
  data/subjects/      # your uploads & generated HTML
